There are a total of 65702
There are 27628 source images

There are an additional 38074 generated images using bounding box augmentations

Augmentations used are brightness [-25 % , 25 %],
bounding box rotation[-6 degree, +6 degree],
bounding box shear[+- 5 degree vertical, +-5 degree horizontal],
bounding box exposure [-15%, 15%].

The images are auto-oriented, static cropped by 8% and resized to 640 x 640.
